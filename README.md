# SleepCycle Pro PWA

Calculadora de ciclos de sueño premium optimizada para dispositivos móviles (Android/iOS).

## Características
- **Cálculo de Despertar**: Encuentra la hora ideal para despertar basándote en ciclos de 90 min.
- **Cálculo de Bedtime**: Encuentra cuándo ir a la cama para despertar renovado.
- **Modo Siesta (Nap)**: Planes rápidos de 20, 30 y 90 minutos.
- **PWA Ready**: Instala la app en tu móvil directamente desde el navegador.
- **Diseño Cosmic-Gray**: Temas oscuros premium con glassmorphism.

## Tecnologías
- HTML5 / CSS3 (Vanilla)
- JavaScript (Vanilla)
- PWA (Service Workers & Manifest)
- Google Fonts (Outfit)

## Instalación en Android
1. Abre la URL en Chrome.
2. Toca los tres puntos (menú).
3. Selecciona **"Instalar aplicación"** o **"Añadir a pantalla de inicio"**.
