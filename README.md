# SleepCycle Pro (OneUI 8 Edition)

Premium Sleep Cycle Planner optimized for **Samsung OneUI 8** devices.

## Features
- **Liquid Glass Aesthetic**: Modern AMOLED-black design with extreme frosting effects.
- **Smart Alarm Shortcut**: Copy sleep times and launch the Clock app with one tap.
- **Now Bar**: Real-time sleep estimate if you go to bed right now.
- **Persistent History**: Local sleep tracking using IndexedDB.
- **PWA Ready**: Installable on Android home screens with full offline support.

## Tech Stack
- HTML5 / CSS3 (Vanilla)
- Vanilla Javascript
- IndexedDB for Persistence
- Service Workers (PWA)
